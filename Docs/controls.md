# Controls Guide

This page covers the current and planned control methods for Corndog.

## Current control methods

### Steam Deck control

Launch with:

```bash
python3 SteamDeckCommunication.py
```

### Local control / experimentation

Launch with:

```bash
python3 main.py
```

### Flipper Zero control

Flipper Zero support is included in the project and is currently considered effectively working.
